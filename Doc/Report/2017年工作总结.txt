CnPack 开发组 2017 年工作总结

    CnPack 在 2017 年取得的主要成绩如下：

一、CnPack IDE 专家包连续发布 1.1.2、1.1.3 版本，增加对 RAD Studio 10.2 Tokyo 的支持。

    每日构建版下载地址：
    http://www.cnpack.org/downbuilds.php

二、CnPack 源码继续于 GitHub 上维护。地址：http://github.com/cnpack

三、CnVcl 新增部分单元，包括 HMAC、SHA3 等，并扩展了 CnDebug、网络格式定义等单元。

四、CnPack 公益基金的帐户继续公开接受用户捐助，目前支持支付宝扫码与微信扫码两种主要捐助方式。

    链接：
    http://www.cnpack.org/foundation.php
    公开帐目：
    http://www.cnpack.org/showdetail.php?id=529

    2018 年 CnPack 开发组的计划如下：

一、CnWizards 统一更换新风格图标。继续增强功能、修复 Bug，增加对新版本 IDE 的支持。

二、CnWizards 将进一步加强国内外的宣传工作。希望能继续进一步扩大用户群。

三、继续扩充及移植 CnPack 组件包，补充和完善各类文档。

四、继续进行 CnPack 开发组的团队建设与规模发展。
