CnPack 开发组 2022 年工作总结

    CnPack 在 2022 年取得的主要成绩如下：

一、CnPack IDE 专家包连续发布 1.2.1、1.2.2、1.2.3、1.2.4、1.2.5 版本，增加对 RAD Studio 11（包括 11.2）的支持。

    每日构建版下载地址：
    http://www.cnpack.org/downbuilds.php

二、CnPack 源码继续于 GitHub 上维护。地址：http://github.com/cnpack

三、CnVcl 中基础库新增了对 GB18030 编码的支持，密码库中新增了 25519 椭圆曲线的实现，128 位长整型运算支持，并完善对 Win32/Win64/MacOS/Linux 等的跨平台支持。增加密码库的 FMX 与 FPC 的支持。网络库部分组件新增跨平台支持。

四、CnPack 公益基金的帐户继续公开接受用户捐助，目前支持支付宝扫码与微信扫码两种主要捐助方式。

    链接：
    http://www.cnpack.org/foundation.php
    公开帐目：
    http://www.cnpack.org/showdetail.php?id=529

    2023 年 CnPack 开发组的计划如下：

一、CnWizards 继续增强功能、修复 Bug，增加对未来可能的新版本 RAD Studio 12 及后续版本的支持。

二、CnWizards 将进一步加强国内外的宣传工作。希望能继续进一步扩大用户群。

三、继续扩充及移植 CnPack 组件包，补充和完善各类文档。

四、继续进行 CnPack 开发组的团队建设与规模发展。

CnPack Working Report in 2022

    Below are the achievements of CnPack in 2022:

1. CnPack IDE Wizards 1.2.1/1.2.2/1.2.3/1.2.4/1.2.5 published. Support RAD Studio 11, including 11.2.

   Our nightly build version:
   http://www.cnpack.org/downbuilds.php

2. CnPack Sources are maintained at GitHub. https://github.com/cnpack

3. CnVcl Supports GB18030, ECC 25519, Int 128. Crypto Library Supports Win32/Win64/MacOS/Linux with FMX/FPC. Network Library Suppors FMX.

4. CnPack Foundation accepts Donation using alipay and wechat QR code in our help files, and paypal: donation@cnpack.org

    Below is the plan of CnPack for 2023:

1. Continue to fix the bugs and do the function improvements for CnWizards. We hope users number can increase considerably in the next year.

2. Continue maintaining CnVcl library and documentation.

3. Continue to build the CnPack Team.
